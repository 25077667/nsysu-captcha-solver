# 中山大學驗證碼破解器

這是一個專門給解析中山大學驗證碼的機器學習專案。

# 如何使用
1. 安裝於 Chrome 擴充套件。
2. 可以存取并修改網頁 -> 當你點擊此擴充套件。

# 如何安裝
## 安裝擴充套件
1. 下載 extension.zip
2. 解壓縮
3. Chrome > 設定 > 擴充套件 > 開發人緣模式 > 選擇解壓縮之資料夾(需含有`manifest.json`之目錄)
